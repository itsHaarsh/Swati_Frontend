# SWATHI - E-commerce Frontend

Health & Cosmetics e-commerce platform built with React + Vite + Strapi.

## Setup

```bash
npm install
npm run dev
```

## Environment

Create `.env`:
```
VITE_API_URL=http://localhost:1337
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS + shadcn/ui
- Strapi Backend Integration

## Features

- Product catalog with search
- Shopping cart (localStorage)
- User authentication
- Checkout with address
- Order management
- Email notifications
